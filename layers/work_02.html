WORK 02

디지털성범죄피해자지원센터 삭제지원시스템 고도화

조건 기반 통계, 문서 자동화, 전송 데이터 보안 강화를 중심으로 고도화를 수행한 프로젝트입니다. 기능 설명뿐 아니라 실제 업무 흐름 관점으로도 정리하기 좋도록 구성했습니다.

주요 수행 작업

사용자 입력 조건 기반 통계 집계 및 조회 기능 구현
불법성 증명 공문 관리 및 문서 자동화 기능 개발
토큰 기반 암호화 적용으로 Form 전송 데이터 보안 강화

키워드

Node.js Express MariaDB 통계 집계 문서 자동화 보안 강화

정리 방향

사용자 입력 조건이 SQL 집계 결과로 이어지는 흐름
공문 생성/관리 프로세스와 자동화 포인트
토큰 기반 암호화 적용 위치와 보호 대상 데이터
운영 효율성과 보안성 측면의 개선 효과

추가 예정

통계 화면 흐름, 집계 쿼리 설명, 문서 자동화 예시, 암호화 처리 코드와 적용 위치를 여기에 정리할 수 있습니다.
특히 이 프로젝트는 사용자 조건 기반 통계와 보안 강화를 함께 보여줄 수 있어, 기능 구현과 실무 안정성 두 측면을 동시에 드러내기 좋은 레이어입니다.
