수행 작업 상세

유관기관 DB화 + 지도API 연동

기존에 정적으로 관리되던 기관 정보를 DB 기반으로 전환하고, 지도 API와 연동하여 활용도를 높인 내용을 정리하는 영역입니다. 데이터 구조 변경 이유와 지도 표시 흐름을 함께 보여주면 좋습니다.

이후 추가 예정 예시: 기관 데이터 구조, 관리 방식 개선, 지도 연동 흐름, 사용자 편의성 향상
실시간 알림 기능 (Socket.io)

사용자 또는 운영자에게 실시간으로 상태를 전달하는 알림 기능을 구현한 내용을 정리하는 영역입니다. 어떤 이벤트가 발생했을 때 어떤 사용자에게 어떤 방식으로 전달되는지 쓰기 좋습니다.

이후 추가 예정 예시: 이벤트 발생 조건, 소켓 연결 흐름, 알림 표시 방식, 운영상 효과
Dropzone 파일 라이브러리 연동

파일 업로드 UX를 개선하기 위해 Dropzone 라이브러리를 연동한 내용을 정리하는 영역입니다. 기존 업로드 방식과 비교해 무엇이 좋아졌는지 함께 적으면 전달력이 높아집니다.

이후 추가 예정 예시: 기존 업로드 문제점, Dropzone 도입 이유, UI 개선 포인트, 사용자 경험 변화
