WORK 01

사이버범죄신고시스템 고도화 및 유지보수

외부 연계, 대용량 데이터 처리, 파일 관리 성능 개선까지 폭넓게 수행한 프로젝트입니다. 이후 이 영역에 전후 비교, 핵심 코드, 처리 흐름을 계속 확장할 수 있도록 기본 구조를 분리했습니다.

주요 수행 작업

외부 제공 API 개발
대용량 데이터 쿼리 성능 최적화
이기종 DB(MariaDB, MongoDB) 간 데이터 연계 및 적재 수행
파일 관리 시스템 성능 최적화

키워드

Java Spring MariaDB MongoDB API 성능 최적화

정리 방향

API 제공 배경과 요청/응답 구조
병목 구간 식별 과정과 튜닝 포인트
DB 간 연계 흐름과 적재 방식
파일 관리 성능 개선 전후 비교

추가 예정

전후 비교 표, 쿼리 개선 포인트, 적재 흐름도, 관련 코드 조각을 이 영역에 이어서 넣을 수 있습니다.
이 레이어는 프로젝트 상세를 파일 단위로 분리한 구조입니다. 그래서 이후 내용이 길어져도 index.html을 복잡하게 만들지 않고, 프로젝트별로 독립적으로 확장할 수 있습니다.
